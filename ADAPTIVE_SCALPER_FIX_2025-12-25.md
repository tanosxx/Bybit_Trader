# Adaptive Scalper Fix - Борьба с "Вялым Сползанием"

**Дата:** 25 декабря 2025, 19:00 UTC  
**Проблема:** Бот открывает сделки в зоне CHOP 50-60 и держит их 180 минут до TTL  
**Решение:** Сдвиг порогов CHOP + снижение TTL для быстрого выхода

---

## 📊 Анализ проблемы

### Текущая ситуация (за 24 часа)
- **Сделок:** 9
- **Wins:** 1 (11%)
- **Losses:** 3 (33%)
- **Zombie trades:** 7 из 9 (78%) - закрыты по TTL
- **PnL:** -$4.65
- **Средняя длительность:** 180 минут (максимум)

### Корень проблемы
**CHOP Index 50-60 = "Вялое сползание"**
- Бот считает это TREND режимом (CHOP < 60)
- Открывает позиции с ожиданием сильного движения
- Цена двигается медленно, не доходит ни до TP, ни до SL
- Zombie Killer закрывает через 180 минут в ноль или минус

**Пример:**
```
SOLUSDT SELL @ $121.61
- Ожидание: сильный даунтренд (CHOP 54)
- Реальность: медленное падение $121.61 → $122.16
- Результат: -$1.65 через 180 минут (TTL)
```

---

## 🔧 Изменения в config.py

### 1. CHOP Пороги (Определение режима)

**БЫЛО:**
```python
chop_flat_threshold: float = 62.0  # CHOP >= 62 = FLAT
chop_trend_threshold: float = 55.0  # CHOP <= 55 = TREND
# Зона 55-62 = гистерезис (сохраняем текущий режим)
```

**СТАЛО:**
```python
chop_flat_threshold: float = 50.0  # CHOP >= 50 = FLAT (было 62)
chop_trend_threshold: float = 45.0  # CHOP <= 45 = TREND (было 55)
# Зона 45-50 = гистерезис (сужена с 55-62)
```

**Эффект:**
- CHOP 50-60 теперь = **FLAT режим** (Adaptive Scalper)
- Активируется Mean Reversion + короткие TP/SL
- Только CHOP < 45 = сильный тренд (TREND режим)

### 2. TTL (Time To Live)

**БЫЛО:**
```python
max_hold_time_minutes: int = 180  # 3 часа
# Adaptive: FLAT → 90 минут (180÷2)
```

**СТАЛО:**
```python
max_hold_time_minutes: int = 120  # 2 часа (было 180)
# Adaptive: FLAT → 60 минут (120÷2), TREND → 120 минут
```

**Эффект:**
- TREND режим: 120 минут (было 180) = -33%
- FLAT режим: 60 минут (было 90) = -33%
- Быстрее выходим из "зависших" сделок

### Сценарий 1: CHOP 50-60 (Вялое сползание)
**Было:**
- Режим: TREND
- TP: 3%, SL: 1.5%
- TTL: 180 минут
- Результат: Зависаем на 3 часа

**Стало:**
- Режим: FLAT (Adaptive Scalper)
- TP: 1.5% (3% × 0.5), SL: 1.05% (1.5% × 0.7)
- TTL: 60 минут
- Результат: Быстрый выход за 1 час

### Сценарий 2: CHOP 45-50 (Гистерезис)
- Сохраняем текущий режим
- Избегаем частых переключений

### Сценарий 3: CHOP < 45 (Сильный тренд)
- Режим: TREND
- TP: 3%, SL: 1.5%
- TTL: 120 минут (было 180)
- Результат: Ловим тренд, но не зависаем надолго
