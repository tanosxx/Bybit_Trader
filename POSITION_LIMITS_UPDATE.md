## 🧠 Решение 2: Self-Learning автообучение

### Проблема:
- Модель загружена с 8949 сэмплами
- **Не обучалась** на новых закрытых сделках
- Predictions: 0 (не использовалась)

### Причина:
Позиции закрывались через:
- `Reversing to LONG/SHORT` (реверс позиции)
- `Sync: closed on exchange` (синхронизация)

А обучение ML было только в `position_monitor.py` (SL/TP), который не использовался.

### Решение:
Добавлено обучение ML в `futures_executor.py` → `_close_position()`:

```python
# После закрытия позиции
if trade.ml_features:
    learner = get_self_learner()
    result = 1 if pnl > 0 else 0  # Win/Loss
    learner.learn(trade.ml_features, result)
    print(f"🧠 Self-Learning: Learned from {'WIN' if result == 1 else 'LOSS'}")
```

### Результат:
- ✅ Модель будет обучаться после **каждой** закрытой сделки
- ✅ Автоматическое улучшение предсказаний
- ✅ Рост `learned_samples` с каждой сделкой
