### 3. Исправлена ошибка получения дневных свечей
**Проблема**: `Failed to get daily candles: 1`
- Код пытался обращаться к свечам как к массиву (`candle[1]`)
- API возвращает словари (`candle['open']`)

**Решение**:
```python
# Было (неправильно):
'open': float(candle[1])

# Стало (правильно):
'open': candle['open']
```

### 4. Гибкая система обновления режима

**Было**: Проверка раз в 4 часа (слишком редко)

**Стало**: Гибридная система
- **Базовый интервал**: 1 час (вместо 4)
- **Триггер на волатильность**: Если BTC изменился на ±3% → обновить сразу
- **Кэширование**: Между проверками используется кэш

**Логика**:
```
Обычный день:
- 10:00 → проверка (SIDEWAYS)
- 10:30 → кэш (SIDEWAYS)
- 11:00 → проверка (SIDEWAYS)

Волатильный день:
- 10:00 → проверка (SIDEWAYS)
- 10:15 → BTC +3.5% → ТРИГГЕР → проверка (BULL_RUSH)
- 10:30 → кэш (BULL_RUSH)
- 11:00 → проверка (BULL_RUSH)
```
