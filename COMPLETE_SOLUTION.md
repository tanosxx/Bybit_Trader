### 2. Конвертация в USD ✅
- BTC конвертируется по текущему курсу с Bybit
- ETH конвертируется по текущему курсу с Bybit
- USDT/USDC = 1:1 к USD

### 3. Правильный расчет PnL ✅

**Начальный баланс:**
- USDT: $50,000
- USDC: $50,000
- BTC: 1.0 × $87,000 = $87,000
- ETH: 1.0 × $2,900 = $2,900
- **ИТОГО: ~$189,900**

**Текущий баланс:**
- USDT: $124,471.32
- USDC: $50,000
- BTC: 0.14270000 × $87,000 = $12,414.90
- ETH: 1.0 × $2,900 = $2,900
- **ИТОГО: ~$189,786**

**PnL: -$114** (небольшой убыток из-за комиссий)

### 4. Синхронизация работает ✅
- Каждые 30 секунд обновляет данные с Bybit API
- Записывает в БД для истории

### 5. ML система работает ✅
**Файл:** `core/price_predictor.py`

**Что делает:**
- Анализирует исторические свечи
- Предсказывает направление цены (UP/DOWN/NEUTRAL)
- Предсказывает цену через 5 и 15 минут
- Использует скользящие средние и линейную регрессию

**НО:** Не обучается на исторических данных (простая модель)
