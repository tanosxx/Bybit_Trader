### 2. futures_brain.py - Строгие агенты

**Пороги агентов:**
```python
# БЫЛО (агрессивно):
'conservative': {'min_confidence': 55, 'require_ta': True}
'balanced': {'min_confidence': 35, 'require_ta': False}
'aggressive': {'min_confidence': 25, 'require_ta': False}
self.min_score_to_trade = 1  # Достаточно 1 агента

# СТАЛО (Safe Mode):
'conservative': {'min_confidence': 70, 'require_ta': True}  # Только отличные
'balanced': {'min_confidence': 60, 'require_ta': True}  # Требуем TA!
'aggressive': {'min_confidence': 50, 'require_ta': False}  # Минимум средние
self.min_score_to_trade = 3  # Нужен консенсус!
```

**Что это значит:**
- Conservative голосует только при 70%+ confidence + TA подтверждение
- Balanced голосует при 60%+ confidence + TA подтверждение
- Aggressive голосует при 50%+ confidence
- Для входа нужно Score >= 3:
  - Conservative (3) ✅
  - Balanced (2) + Aggressive (1) = 3 ✅
  - Только Aggressive (1) ❌ недостаточно!

## 📊 Мониторинг

### SQL запросы для проверки

**Win Rate за 24 часа:**
```sql
SELECT 
    COUNT(*) FILTER (WHERE pnl > 0) as wins,
    COUNT(*) as total,
    ROUND(COUNT(*) FILTER (WHERE pnl > 0)::numeric / COUNT(*) * 100, 1) as win_rate
FROM trades 
WHERE status = 'CLOSED' 
  AND market_type = 'futures' 
  AND entry_time > NOW() - INTERVAL '24 hours';
```

**Avg Net PnL:**
```sql
SELECT 
    ROUND(AVG(pnl - (fee_entry + fee_exit))::numeric, 2) as avg_net_pnl,
    COUNT(*) as trades
FROM trades 
WHERE status = 'CLOSED' 
  AND market_type = 'futures' 
  AND entry_time > NOW() - INTERVAL '24 hours';
```

**Текущий баланс:**
```sql
SELECT 
    100.0 + SUM(pnl) - SUM(fee_entry + fee_exit) as balance
FROM trades 
WHERE status = 'CLOSED' 
  AND market_type = 'futures';
```
