# 🔧 Hybrid Strategy Hysteresis Fix - 14 декабря 2025

## ❌ Проблема

**Симптомы:**
- Dashboard показывает частые переключения между FLAT и TREND
- Режим скачет туда-сюда с небольшой разницей
- Бот ничего не торгует

**Причина:**
CHOP индекс колеблется около порога 60:
- CHOP 59.5 → TREND
- CHOP 60.1 → FLAT
- CHOP 59.8 → TREND
- CHOP 62.7 → FLAT

Каждое переключение сбрасывает логику, и бот не успевает найти сигнал.

**Дополнительная проблема:**
- В TREND режиме: ML confidence 26-30% (ниже минимума 60%)
- В FLAT режиме: RSI neutral (50-62), ждёт экстремумов (< 30 или > 70)

---

## ✅ Решение: Гистерезис (Hysteresis)

### Концепция
Вместо одного порога (60), используем два:
- **Переход в FLAT:** CHOP >= 62
- **Переход в TREND:** CHOP <= 58
- **Зона гистерезиса:** 58-62 (сохраняем текущий режим)

### Как Работает

```
CHOP < 58     →  TREND (однозначно)
CHOP 58-62    →  Сохраняем предыдущий режим (гистерезис)
CHOP > 62     →  FLAT (однозначно)
```

**Пример:**
1. CHOP = 55 → TREND (< 58)
2. CHOP = 59 → TREND (зона гистерезиса, сохраняем TREND)
3. CHOP = 60 → TREND (зона гистерезиса, сохраняем TREND)
4. CHOP = 61 → TREND (зона гистерезиса, сохраняем TREND)
5. CHOP = 63 → FLAT (>= 62, переключаемся)
6. CHOP = 60 → FLAT (зона гистерезиса, сохраняем FLAT)
7. CHOP = 57 → TREND (<= 58, переключаемся)

**Стало:**

```python
# CHOP пороги с гистерезисом (избегаем частых переключений)
chop_flat_threshold: float = 62.0  # CHOP >= 62 = переход в FLAT
chop_trend_threshold: float = 58.0  # CHOP <= 58 = переход в TREND
# Зона 58-62 = сохраняем текущий режим (гистерезис)
```

### 2. core/ai_brain_local.py

**Добавлено:**
```python
# Читаем предыдущий режим для гистерезиса
previous_mode = 'TREND'  # По умолчанию TREND
try:
    import json
    import os
    if os.path.exists('/app/ml_data/hybrid_strategy_state.json'):
        with open('/app/ml_data/hybrid_strategy_state.json', 'r') as f:
            prev_state = json.load(f)
            previous_mode = prev_state.get('market_mode', 'TREND')
except:
    pass

# Определяем режим рынка по CHOP с гистерезисом
if chop >= settings.chop_flat_threshold:
    # CHOP >= 62 → однозначно FLAT
    market_mode = 'FLAT'
elif chop <= settings.chop_trend_threshold:
    # CHOP <= 58 → однозначно TREND
    market_mode = 'TREND'
else:
    # CHOP в зоне 58-62 → сохраняем предыдущий режим
    market_mode = previous_mode
    print(f"   ⚖️  Hysteresis Zone (CHOP: {chop:.1f}) - keeping {market_mode} mode")
```

## ⚠️ Оставшиеся Проблемы

### 1. Низкая ML Confidence (26-30%)
**Причина:** Рынок неопределённый (SIDEWAYS режим Strategic Brain)

**Решение:** Это нормально! Консервативная фильтрация работает правильно.
- ML confidence < 60% → SKIP
- Лучше пропустить сделку, чем потерять деньги

### 2. Mean Reversion Ждёт Экстремумов
**Причина:** RSI neutral (50-62), нет перепроданности/перекупленности

**Решение:** Это тоже нормально! Mean Reversion работает только на экстремумах:
- RSI < 30 → BUY signal
- RSI > 70 → SELL signal
- RSI 30-70 → SKIP (ждём)

### 3. Нет Сделок
**Причина:** Комбинация факторов:
- Strategic Brain: SIDEWAYS (неопределённость)
- ML Confidence: 26-30% (низкая)
- RSI: 50-62 (нейтральный)
- CHOP: 58-62 (переходная зона)

**Решение:** Ждать! Когда рынок определится:
- Либо CHOP < 58 + ML confidence > 60% → TREND сделки
- Либо CHOP > 62 + RSI < 30 или > 70 → FLAT сделки

## 🎓 Теория: Что Такое Гистерезис?

**Гистерезис** - это явление, когда система имеет "память" о предыдущем состоянии.

**Примеры из жизни:**
1. **Термостат:** 
   - Включается при 18°C
   - Выключается при 22°C
   - Зона 18-22°C = гистерезис (избегает частых включений/выключений)

2. **Автоматические двери:**
   - Открываются при приближении на 1 метр
   - Закрываются при удалении на 2 метра
   - Зона 1-2 метра = гистерезис (избегает дрожания)

3. **Наш бот:**
   - TREND → FLAT при CHOP >= 62
   - FLAT → TREND при CHOP <= 58
   - Зона 58-62 = гистерезис (избегает частых переключений)

**Преимущества:**
- Стабильность системы
- Меньше шума
- Экономия ресурсов
- Лучшая производительность
