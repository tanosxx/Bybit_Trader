## 🔧 Технические детали

### Bybit API интеграция

Dashboard использует:
- `/v5/account/wallet-balance` - баланс кошелька
- `/v5/position/list` - открытые позиции
- `accountType: UNIFIED` - Unified Trading Account
- `category: linear` - для фьючерсов (если используем)
- `category: spot` - для спотовой торговли

### Синхронная обертка

Создан класс `BybitAPISync` для работы в Streamlit:
- Синхронные методы для асинхронного API
- Автоматическая подпись запросов (HMAC SHA256)
- Поддержка Testnet и Mainnet

### Автообновление

Dashboard обновляется каждые 10 секунд:
- Реальный баланс через API
- Реальные позиции через API
- История сделок из БД
- Логи из БД
