### Что такое "фантомные закрытия"?

Это позиции которые были закрыты **Sync Service**, а не ботом:
- `exit_reason = "Sync: not found on exchange"` - позиция исчезла с биржи
- `exit_reason = "Sync: reversed to BUY/SELL"` - сторона изменилась
- `pnl = 0` и `exit_price = entry_price` - закрыто без движения цены

**Причины:**
1. Бот открыл позицию, но она была закрыта вручную на бирже
2. Позиция закрылась по SL/TP на бирже, но бот не успел обновить БД
3. Sync Service нашёл расхождение и закрыл в БД

### Реальные сделки (125 шт)

```sql
SELECT 
    COUNT(*) as real_trades,
    SUM(CASE WHEN pnl > 0 THEN 1 ELSE 0 END) as wins,
    SUM(CASE WHEN pnl < 0 THEN 1 ELSE 0 END) as losses,
    ROUND((SUM(CASE WHEN pnl > 0 THEN 1 ELSE 0 END)::float / COUNT(*)::float * 100)::numeric, 1) as win_rate,
    ROUND(SUM(pnl)::numeric, 2) as gross_pnl,
    ROUND(SUM(fee_entry + fee_exit)::numeric, 2) as total_fees,
    ROUND((SUM(pnl) - SUM(fee_entry + fee_exit))::numeric, 2) as net_profit
FROM trades 
WHERE status = 'CLOSED' 
  AND exit_reason NOT LIKE 'Sync:%' 
  AND pnl != 0;
```
